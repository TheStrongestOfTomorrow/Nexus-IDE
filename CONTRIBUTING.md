## 🚀 Quick Start

```bash
# 1. Fork & Clone
git clone https://github.com/YOUR_USERNAME/Nexus-IDE.git
cd Nexus-IDE

# 2. Install
npm install

# 3. Create Branch
git checkout -b feature/my-awesome-feature

# 4. Make Changes & Test
npm run dev

# 5. Commit & Push
git commit -m "feat: my awesome feature"
git push origin feature/my-awesome-feature

# 6. Open PR 🎉
```

## 🎯 Branch Guide

| Branch | Purpose | Base For PRs |
|--------|---------|--------------|
| `stable` | Production | Bug fixes only |
| `main` | Beta | Features |
| `cli` | Terminal | TUI improvements |
| `professional` | Enterprise | Office features |

---

## 📝 Commit Convention

```
feat: Add new feature
fix: Fix a bug
docs: Update documentation
style: Format code
refactor: Refactor code
test: Add tests
chore: Maintenance
```

---

## ✅ PR Checklist

- [ ] Code compiles without errors
- [ ] Tested locally
- [ ] Updated documentation
- [ ] Follows code style
- [ ] Descriptive commit messages
