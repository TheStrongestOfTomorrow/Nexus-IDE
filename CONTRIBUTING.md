## 🚀 Quick Contribution

```bash
# 1. Fork & Clone
git clone https://github.com/YOUR_USERNAME/Nexus-IDE.git
cd Nexus-IDE && git checkout professional

# 2. Install
npm install

# 3. Create Branch
git checkout -b feature/pro-feature

# 4. Develop
npm run dev

# 5. Submit PR
git push && # Open PR on GitHub
```
