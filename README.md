## 📋 Commands

```bash
nexus-pro              # Interactive mode selector
nexus-pro tui          # Terminal mode
nexus-pro web          # Web mode
nexus-pro --port 8080  # Custom port
nexus-pro ai "prompt"  # Quick AI question
```
